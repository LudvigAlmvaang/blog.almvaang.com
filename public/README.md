# Public Assets

This folder contains static assets for the blog:

- `favicon.svg` - Browser tab icon
- `icons/` - SVG icons
- `images/` - Images used in posts or UI

## Usage

Reference these files in markdown or HTML:

```markdown
![My Image](/images/my-image.png)
```

In HTML/JavaScript:
```html
<img src="/images/my-image.png" alt="My Image">
```
